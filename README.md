# GenAI MLOps Starters

Compact public reference projects for GenAI, MLOps, and restricted enterprise environments.

## Projects

### 1. Banking Ops RAG Assistant (2025)
Minimal FastAPI-based Retrieval-Augmented Generation (RAG) assistant pattern with `/health` and `/ask` endpoints.

### 2. ML Lifecycle on Azure ML (2025)
Minimal machine learning lifecycle example showing local training, MLflow metrics, and reproducible experiment structure.

### 3. Air-Gapped GenAI Pattern (2025)
Reference architecture pattern for preparing GenAI workloads for restricted environments using private registries and controlled image mirroring.

## Important note

These projects are intentionally compact public demos. They do not contain confidential enterprise data, real banking architecture, private IP addresses, hostnames, customer information, or production configuration.
